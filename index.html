How it works

The model plans. Code verifies and computes.

This keeps the language model’s job small and inspectable. It cannot quietly replace a calculation with a plausible sentence.

  1. 01

    Ask naturally

    Request a value, comparison, ranking, table, or plot.

    “Average ozone in Bengaluru”
  2. 02

    Build a small plan

    A fine-tuned local model selects the operation, filters, metric, and output.

    filter · group · mean · table
  3. 03

    Check the request

    Unknown cities, pollutants, dates, and malformed plans are rejected before execution.

    schema ✓ scope ✓ range ✓
  4. 04

    Run on local data

    Native Swift or Kotlin code computes the answer and renders the result.

    answer · table · line · bar

Research

Benchmarked and published.

VayuBench evaluates whether a model can produce an executable program and the correct answer—not merely fluent text.

Primary paper · CODS 2025

VayuBench and VayuChat: Executable Benchmarking and Deployment of LLMs for Multi-Dataset Air Quality Analytics

Vedant Acharya, Abhay Pisharodi, Ratnesh Pasi, Rishabh Mondal, and Nipun Batra

The benchmark contains 5,000 natural-language queries paired with verified programs across seven query families. The study evaluates 13 open models under a common schema-aware protocol and follows the work through to VayuChat.
